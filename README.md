### 3.2-MVC-books-library

Program uses .env variables so in order to run please set them up:
```
PORT=***
DB_HOST="***"
DB_USER="***"
DB_PASSWORD="***"
DB_DATABASE="***"
```
In order to compile the code please use:
```
npm run build
```
After compiling you can start app using:
```
npm run start
```
